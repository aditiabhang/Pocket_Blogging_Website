About


Pocket Blog Website is developed during Summer 2020 as a project course requirement for the course CS5394 - Advanced Software Engineering Project.
The website is a full featured content management project developed with the following specifications and tools:

TOOLS:
Programming language : Python
Development Framework : Flask
IDE : PyCharm

FRONT-END:
HTML5, CSS3, Bootstrap 4 (embedded with Jinja2 template).

BACK-END
Database : SQLAlchemy & SQLite (Development), PostgreSQL (Deployment)
Deployed on : HerokuApp
